## Instructions

This library can be found on [NPM](https://www.npmjs.com/package/izui-react)

```sh
npm install izui-react
```

## Building (Any Platform)

_Pre-Requisites:_
1. [NodeJS (version 14+)](https://nodejs.org/en/)
2. [Yarn / NPM](https://yarnpkg.com/)
